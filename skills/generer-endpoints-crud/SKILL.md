\# SKILL: generer-endpoints-crud



\## Entrées attendues

\- Nom de l'entité (ex: "Product")

\- Champs du modèle (ex: name, price, stock)



\## Outils à utiliser

\- Spring Boot

\- Spring Data JPA



\## Résultat attendu

\- Controller REST avec les 5 endpoints standards (GET all, GET by id, POST, PUT, DELETE)

\- Service + Repository associés

\- Codes de statut HTTP corrects (200, 201, 404, etc.)



\## Procédure

1\. Créer l'entité JPA avec les champs demandés

2\. Créer le Repository (interface JpaRepository)

3\. Créer le Service avec la logique métier

4\. Créer le Controller avec les 5 endpoints

5\. Vérifier que chaque endpoint retourne le bon code HTTP
